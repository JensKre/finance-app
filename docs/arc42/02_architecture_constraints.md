# 2. Architecture Constraints

## 2.1 Technical Constraints
- **Language**: JavaScript (Frontend) and Python (Backend).
- **Frontend Framework**: React with Vite.
- **Backend Framework**: FastAPI.
- **Persistence**: Flat-file JSON (`data.json`).
- **Operating System**: Optimized for macOS (Safari).

## 2.2 Organizational Constraints
- **Private Use**: Only for Jens and Annika.
- **Environment Separation**: Local development environment must be strictly separated from the "Live" production folder.
- **Deployment**: Manual build and deploy via shell scripts.

## 2.3 Conventions
- **Language**: UI and Documentation in German.
- **Styling**: Vanilla CSS for flexibility.
- **Icons**: Lucide-react.
- **Charts**: Recharts.
