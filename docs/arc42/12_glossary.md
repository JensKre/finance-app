# 12. Glossary

| Term | Definition |
|---|---|
| Account | A unique combination of Person, Institution, and Category. |
| Sparrate | Monthly savings amount used for prognosis. |
| Wachstum | Expected yearly growth rate in percent p.a. |
| Unified Server | A mode where FastAPI serves both its API and the built Frontend assets. |
| latestByInstitute | Logical grouping key used to calculate the current balance of an account. |
