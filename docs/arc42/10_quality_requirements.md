# 10. Quality Requirements

## 10.1 Quality Tree
- **Reliability**: 100% calculation accuracy for financial data.
- **Ease of Use**: Fast data entry (optimized for keyboard/dropdowns).
- **Efficiency**: Near-instant load times for the local dashboard.

## 10.2 Quality Scenarios
| Scenario | Attribute | Priority |
|---|---|---|
| User adds a new record and the total wealth updates immediately. | Responsiveness | High |
| A deployment is run and the existing live data is preserved. | Reliability | High |
| User enters growth with a German comma (e.g. 5,5) and it is parsed correctly. | Usability | Medium |
